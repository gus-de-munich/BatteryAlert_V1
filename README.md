# BatteryAlert_V1
Have you ever been disappointed by a mac app? Because I have... It always starts well, and all of a sudden it needs a subscription...

This is why I decided to take the matter into my own hands and so I made the app myself! For free! (You're welcome)

This app allows you to simply set custom alarms for custom battery levels (simple right?). But that's not all! It also allows you to change the color of the battery icon in the menu bar. 

You can set custom styles (fixed custom color, custom gradient...). So, have fun, it's 100% free!

To download:
1) use .dmg file (you might have to go in the settings to allow the app to launch)
2) use .zip file
3) use install.sh file and paste "chmod +x ~/Downloads/install.sh && ~/Downloads/install.sh" in you terminal
